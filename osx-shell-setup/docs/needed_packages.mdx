# Required Packages for Dotfiles Setup

## Prerequisites

Before setting up the dotfiles, ensure you have the following installed:

- [Homebrew](https://brew.sh) (Package Manager for macOS)
- [Git](https://git-scm.com)
- [Zsh](https://www.zsh.org) (Should be default on modern macOS)

## Core Dependencies

```bash
# Install these first as they're required for core functionality
brew install zsh
brew install git
```

## Plugin Dependencies

These packages are required for various plugin functionalities:


## Plugin Installation

The plugins are managed as Git submodules in the dotfiles repository:

- zsh-autosuggestions
- zsh-syntax-highlighting
- zsh-completions
- powerlevel10k
- z
- fzf

These are automatically installed when cloning the dotfiles repository with submodules:

```bash
git clone --recurse-submodules https://github.com/yourusername/dotfiles.git
```

## Post-Installation

After installing all packages:

1. Restart your terminal
2. Run `p10k configure` to set up your Powerlevel10k theme
3. Configure development environments:
   - `fnm use --lts` to set Node.js version
   - `pyenv install 3.11` for Python
   - Set up Docker Desktop preferences
4. Verify that all plugins are working:
   - Type a command to see autosuggestions
   - Use CTRL+T to test fzf
   - Check if syntax highlighting is working
   - Try tab completion

## Maintenance

Keep your tools updated regularly:

```bash
# Update Homebrew and all packages
brew update
brew upgrade

# Update Node.js packages
pnpm update -g

# Update Python packages
pip3 install --upgrade pip
pip3 list --outdated

# Update Rust
rustup update

# Update all Git submodules (plugins)
git submodule update --remote --merge
```
